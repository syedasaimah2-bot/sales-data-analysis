# 📊 Sales Data Analysis Project

## 📌 Project Overview
This project analyzes sales data using Python to generate business insights such as total revenue and top-performing products. It also includes data visualization using charts.

---

## 🚀 Features
- Load sales data from CSV file
- Calculate total revenue
- Identify best-selling products
- Perform data analysis using Python
- Visualize data using charts 📊

---

## 🛠️ Tools & Technologies
- Python
- Pandas
- Matplotlib

---

## 📂 Project Structure
sales-data-analysis/
│── analysis.py
│── sales.csv
│── README.md
│── chart.png (screenshot of output)

---

## ▶️ How to Run

1. Install required libraries:
pip install pandas matplotlib

2. Run the project:
python analysis.py
